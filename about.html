使用方式

  1. 前往 rent.591.com.tw 或 sale.591.com.tw 的物件詳情頁
  2. 點擊瀏覽器右上角的 HomeScope 圖示 開啟插件
  3. 點選生活機能按鈕（捷運站、全聯、便利商店等）在 Google Maps 中搜尋
  4. 收藏物件後，可在「看房清單」管理看房狀態、筆記與評分
  5. 勾選多筆物件，前往「比較」頁籤進行規格對比
  6. 在 ⚙️ 設定 中新增常去地點，搜尋頁將自動顯示通勤距離

資料來源與計算方式

🌋 土壤液化潛勢

資料來源：中央地質調查所（liquid.net.tw）。依物件座標即時查詢，回傳高 / 中 / 低潛勢等級，無資料時顯示灰色。屬全台通用資料，精確度至區域等級。

🌊 淹水潛勢

資料來源：經濟部水利署 GIS 圖台（Layer 116，24 小時 650mm 最嚴峻情境）。以物件座標查詢所在格網，回傳淹水深度等級（1–5 級），無覆蓋區域視為無淹水潛勢。

⚡ 活動斷層距離

資料來源：中央地質調查所地質雲（134 條活動斷層 GeoJSON）。下載後快取 30 天，在本機計算物件座標至每條斷層線段的最短球面距離（Haversine 公式），標示最近斷層名稱與距離。

📋 社區實價登錄溢價分析

資料來自內政部實價登錄，經 591 市場 API 取得，本機計算溢價百分比。

透天厝等無社區 ID 之物件不顯示 · 依社區均價比對，非同棟成交

💰 貸款試算

純本機計算，無外部資料。以本利平均攤還（PMT）公式即時試算月付金額。

可調整成數（六 / 七 / 八成）、年期（20 / 30 年）、利率（2% / 2.5% / 3%）

📉 降價追蹤

直接從 591 物件頁面抓取最新掛牌價，與上次記錄比對顯示漲跌。

看房清單可一鍵批次更新 · 本機保留最多 10 筆歷史

所有資料均為公開來源，HomeScope 不儲存任何個人資訊，亦不上傳查詢結果。
