# 🏠 HomeScope

![HomeScope](imag/440x280.png)

> 在 591 租屋／售屋物件頁面，一鍵掌握周邊生活機能與環境風險的 Chrome 擴充功能。

[![Chrome Web Store](https://img.shields.io/badge/Chrome%20Web%20Store-即將上架-blue)](https://chromewebstore.google.com)

---

## 功能

- **生活機能搜尋**：捷運站、全聯、便利商店、國小、國中、醫院、公園、銀行等 10 項 + 自訂地點
- **嫌惡設施標記**：宮廟、殯儀館、納骨塔、特種行業、加油站、高壓電塔、垃圾場
- **環境風險分析**：土壤液化（地質調查所）、淹水潛勢（水利署）、活動斷層距離（地質雲）
- **價格分析**：社區實價登錄溢價分析、貸款試算（PMT）、降價追蹤
- **看房清單**：收藏物件、看房狀態、筆記、評分、多物件比較
- **通勤距離**：設定常去地點，搜尋頁自動顯示直線距離

## 支援平台

| 平台 | 網址 |
|------|------|
| 591 租屋 | rent.591.com.tw |
| 591 售屋 | sale.591.com.tw |

## 安裝

Chrome Web Store（即將上架）或載入未封裝擴充功能：

1. 下載此 repo 並解壓
2. 開啟 `chrome://extensions/`
3. 啟用「開發人員模式」
4. 點選「載入未封裝項目」，選擇此資料夾
